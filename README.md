# Student Result Management System (SRMS)

## Project Description
The **Student Result Management System (SRMS)** is a simple Python application that allows users to enter student details, record scores for three subjects, calculate total and average scores, and display the student’s result.

This project was developed as part of the **SEN assignment** to demonstrate the practical application of the **Software Development Life Cycle (SDLC)**.

---

## Features
- Add student name
- Input scores for three subjects
- Calculate total score
- Calculate average score
- Display student result

---

## Technology Used
- **Programming Language:** Python
- **Paradigm:** Procedural Programming
- **Version Control:** Git & GitHub

---

## System Functions
The system uses the following functions:
- `add_student()` – Collects student name and scores
- `calculate_result()` – Calculates total and average scores
- `display_result()` – Displays the student result

---

## How It Works
1. User enters the student name.
2. User inputs scores for three subjects.
3. The system calculates the total and average scores.
4. The result is displayed on the screen.

---

## How to Run the Program
1. Ensure Python is installed.
2. Clone or download this repository.
3. Open the project folder.
4. Run the program using:

```bash
python srms.py
